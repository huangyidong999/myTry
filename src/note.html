5.Two-way binding Example:
ngModel is the way in Anguar of two way binding. Here we bind the hero.name property with the HTML text input box, so it can flow between html text box and the hero.name property To use ngModel, we need to import FormsModel to our app.modules.ts file, like this: import { FormsModule } from '@angular/forms'; // <-- NgModel lives here Then add the FormsModule into or metadata's import array: imports: [ BrowserModule, FormsModule ], Again, the explaination above, all is in the file of app.module.ts THIS IS REALLY IMPORTANT: declar the component into module.ts file
